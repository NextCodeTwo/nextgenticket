Introduction

Welcome to the official documentation for the Next Code Two platform! This guide will help you integrate and manage the authentication system, ticketing, and more.

Authentication System

This section explains how to use the authentication system in Next Code Two, powered by Cloudflare Workers, JWT, and cookies.

Getting Started

To get started with the authentication system, follow the steps below:

  1. Create a Cloudflare Worker for authentication.
  2. Set up JWT-based token authentication.
  3. Store tokens securely in cookies.
  4. Configure routes for login and user registration.

Example Authentication Flow

{
  "method": "POST",
  "url": "/api/login",
  "body": {
    "email": "user@example.com",
    "password": "password123"
  }
}

This is an example of the authentication request. The response will include a JWT token.

Ticketing System

This section covers how to manage and interact with the ticketing system.

Creating a Ticket

You can create a new ticket via the following API:

{
  "method": "POST",
  "url": "/api/tickets",
  "body": {
    "subject": "Issue with authentication",
    "description": "Unable to log in to the dashboard."
  }
}

Replying to a Ticket

To respond to an existing ticket, use the following API endpoint:

{
  "method": "POST",
  "url": "/api/tickets/{ticket_id}/reply",
  "body": {
    "message": "We are investigating your issue."
  }
}
